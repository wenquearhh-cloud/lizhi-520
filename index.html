📋 你的经期自查表

  • 周期 21~35 天都正常，平均 28 天
  • 每次持续 2~8 天，血量 20~60ml（超过 80ml 要留意）
  • 轻微不适正常，但痛到影响生活一定要看医生
  • 🩸 血块小于拇指指甲盖 → 正常；更大或频繁 → 去检查
⚠️ 需要去看医生的情况：血量浸透卫生巾 > 1片/小时、周期连续 3 个月紊乱、非经期出血、45 岁前停经 3 个月以上

💊 经期护理要点

  • 卫生巾每 2~4 小时换一次，量多时更勤
  • 每天温水清洗外阴，淋浴代替盆浴
  • 热敷小腹（40℃ 左右热水袋）缓解痛经
  • 痛经厉害可吃布洛芬（饭后服用，不空腹）
  • 多补铁：红肉、动物肝脏、鸭血、菠菜
  • 少碰冰水、咖啡、酒精——它们会加重不适

🧺 宿舍/租房生活小常识

  • 白醋 + 小苏打 = 万能清洁剂（水垢、油污、马桶都行）
  • 衣柜放一块肥皂/香皂 → 衣服一直香香的还防虫
  • 手机掉水里 → 立刻关机，埋进生米里一夜
  • 衣服沾油渍 → 洗洁精直接涂上搓，再正常洗
  • 烧水壶有水垢 → 白醋加水烧开，泡半小时，焕然一新
  • 鞋子磨脚 → 穿之前用吹风机热风吹软，涂护手霜

🩹 家庭急救小知识

  • 烫伤：流动冷水冲 15-20 分钟！不要涂牙膏、酱油
  • 小割伤：清水冲 → 碘伏消毒 → 创可贴（别包太紧）
  • 流鼻血：头前倾！捏住鼻翼 5-10 分钟，别仰头
  • 崴脚：冰敷（不是热敷！），抬高，休息（RICE 原则）

📝 高效学习方法

  • 番茄钟法：专注 25 分钟 → 休息 5 分钟 → 每 4 轮大休息 15 分钟
  • 费曼学习法：假装给一个完全不懂的人讲一遍，讲不清楚的地方就是你没懂的地方
  • 间隔重复：学完 1 天后复习 → 3 天后 → 7 天后 → 30 天后（Anki 卡片帮你自动安排）
  • 主动回忆：合上书，自己默写/复述，比反复翻书有效 3 倍
  • 睡眠巩固：背完东西直接睡觉，大脑会在睡眠中帮你巩固记忆

🚶‍♀️ 日常安全守则

  • 夜间尽量不独行，走大路、有灯的路，不要走小巷捷径
  • 走路不戴耳机，保持对周围环境的警觉
  • 打车坐后排，上车前拍下车牌发给朋友，行程中保持通话或开位置共享
  • 不坐无牌黑车，只用正规平台（滴滴、高德打车等）
  • 陌生人饮料不喝！在酒吧/聚会，杯子不离视线
  • 感觉被尾随 → 往人多的地方走，进便利店/餐厅，打 110
📱 手机紧急功能：iPhone 连按 5 次电源键 → 自动 SOS；安卓在设置搜「紧急 SOS」开启。提前设好紧急联系人！

🌐 网络安全 + 防诈骗

  • 安装 国家反诈中心 APP！真的有用，能拦截很多诈骗电话
  • 朋友圈不要晒位置、机票、身份证、家门钥匙照片
  • 微信设置：关闭「允许陌生人查看十条朋友圈」
  • 任何「刷单」「高薪兼职」「先交钱的」= 诈骗
  • 自称客服退款/公检法办案来电 → 挂掉，自己打官方电话核实
  • AI 换脸视频通话诈骗 → 让对方转头、用手挡脸一下看是否变形
